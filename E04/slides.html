Modelos de Regresi√≥n Lineal

Teor√≠a

Antes de entrar a hablar de que son los Modelos Lineales Generalizados, es importante mencionar qu√© es un modelo de Regresi√≥n Lineal y de qu√© est√° compuesto. En terminos generales, un modelo estad√≠stico consta de un componente aleatorio y un componente sistem√°tico. El componente aleatorio se refiere a la variabilidad en los datos, mientras que el componente sistem√°tico se refiere a la relaci√≥n entre las variables independientes y la variable dependiente.

En los terminos de la notaci√≥n utilizada por Dunn y Smyth (2018) en su libro ‚ÄúGeneralized Linear Models with Examples in R‚Äù, el componente sistem√°tico en estos modelos, asume que el valor esperado de la variable dependiente depende de p variables independientes a trav√©s de una funci√≥n f con una serie de par√°metros de regresi√≥n. Donde la media en la i-√©sima observaci√≥n de la combinaci√≥n lineal de estos p√°rametros es igual a la i-√©sima observaci√≥n de la variable dependiente.

Por otro lado, el componente aleatorio asume que, en la i-√©sima observaci√≥n, la variable dependiente sigue una distribuci√≥n normal con media y varianza constante. En otras palabras se asume que las respuestas se distribuyen normalmente alrededor de la media con una varianza constante. Para los modelos lineales, la funci√≥n \(f\) es la funci√≥n identidad, \(f(x) = x\). Para los modelos no lineales, la funci√≥n \(f\) puede ser cualquier funci√≥n no lineal.

Un modelo de regresi√≥n lineal asume que la respuesta media \(\mu_{i}\) para la observaci√≥n \(i\) depende de \(p\) variables independientes \(x_{1i}\) a \(x_{pi}\), a trav√©s de una funci√≥n \(f\) con una serie de par√°metros de regresi√≥n \(\beta_{j}\) (para \(j =1,0,...q\)). De tal manera que el componente sistem√°tico de una regresi√≥n lineal se puede expresar como:

\[E[y_{i}] = \mu_{i} = f(\beta_{0} + \beta_{1}x_{1i} + ... + \beta_{p}x_{pi})\]

Mientras que el componente aleatorio asume que la variable dependiente \(y_{i}\) sigue una distribuci√≥n normal con media \(\mu_{i}\) y varianza constante \(\sigma^{2}\):

\[y_{i} \sim N(\mu_{i}, \sigma^{2})\]

Modelos Lineales Generalizados

Teor√≠a

Ahora bien, en un Modelo Lineal Generalizado, que es una extensi√≥n del modelo lineal, el componente sistem√°tico asume que la respuesta media para la observaci√≥n i-√©sima depende de p variables independientes, a trav√©s de una funci√≥n g con una serie de par√°metros de regresi√≥n. Esta funci√≥n g es llamada link function y es una funci√≥n monot√≥nica y diferenciable.

Miestras que el componente aleatorio asume que la variable dependiente sigue una distribuci√≥n de una familia espec√≠fica de distribuciones de probabilidad, llamadas Modelos de Dispersi√≥n Exponencial, que incluye distribuciones comunes como la normal, la Poisson, que veremos en un momento, y el binomio como casos especiales.

En un modelo lineal generalizado (GLM, pos sus siglas en ingl√©s), el componente sistem√°tico se puede expresar como:

\[E[y_{i}] = \mu_{i} = g^{(\beta_{0} + \beta_{1}x_{1i} + ... + \beta_{p}x_{pi})}\]

o alternativamente:

\[E[y_{i}] = g(\mu_{i}) = \beta_{0} + \beta_{1}x_{1i} + ... + \beta_{p}x_{pi}\]

Donde \(g()\) es llamada link function. Que es una funci√≥n monot√≥nica y diferenciable (como una funci√≥n logar√≠tmica).

El componente aleatorio se explica asumiendo que la variable dependiente \(y_{i}\) sigue una distribuci√≥n de una familia espec√≠fica de distribuciones de probabilidad, llamadas Modelos de Dispersi√≥n Exponencial (EDM, por sus siglas en ingl√©s).

Caso de aplicaci√≥n

GLMs Poisson

Teor√≠a

Ya hemos dicho que el componente aleatorio de los modelos lineales generalizados de Poisson siguen una distribuci√≥n de su misma familia y que el componente aleatorio se puede expresar como una funci√≥n exponencial. Ahora bien, la funci√≥n de enlace m√°s comunmente utilizada para los modelos lineales generalizados de Poisson es la funci√≥n logar√≠tmica, la cual garantiza que la media sea mayor a cero. Esto facilita la interpretaci√≥n de los coeficientes de regresi√≥n, ya que representan la raz√≥n de cambio en la media por cada unidad de cambio en el predictor correspondiente.

La funci√≥n de enlace m√°s comunmente utilizada para los GMLs de Poisson es la funci√≥n logar√≠tmica, la cual garantiza que \(\mu>0\). Con esto la forma general de un modelo de regresi√≥n Poisson estar√≠a dada por:

\[\begin{cases} & y \sim Poisson(\mu) \hspace{1cm} \text{Componente aleatorio} \\ & log\mu = \beta_{0} + \beta_{1}x_{1} + ... + \beta_{p}x_{p} \hspace{1cm} \text{Componente sistem√°tico} \end{cases}\]

Donde el componente sistem√°tico tambi√©n se puede expresar como:

\[\mu = \exp(\beta_{0} + \beta_{1}x_{1} + ... + \beta_{p}x_{p})\]

o

\[\mu = \exp\beta_{0} \cdot (\exp\beta_{1})^{x_{1}} \cdot ... \cdot (\exp\beta_{p})^{x_{p}}\]

GLMs Poisson

Teor√≠a

En cuanto a la forma de las variables dependientes e independientes en los modelos de regresi√≥n Poisson, es importante tener en cuenta que la variable dependiente debe ser una variable de conteo, mientras que las variables independientes pueden ser cualitativas, cuantitativas o una mezcla de ambas.

Cuando todas las variables independientes son cualitativas, es apropiado resumir los datos en una tabla de contingencia. En este caso, el modelo se denomina modelo de log-odds lineal o modelo de log-lineal, no modelo logar√≠tmico lineal. La distinci√≥n radica en que el modelo relaciona el logaritmo de la raz√≥n de odds con las variables predictoras, no directamente las variables en s√≠.

Si alguna variable independiente es cuantitativa, el modelo sigue denomin√°ndose modelo de regresi√≥n de Poisson. La presencia de variables cuantitativas no altera la naturaleza fundamental del modelo.

Cuando hay una mezcla de variables cualitativas y cuantitativas como variables independientes, se puede utilizar un modelo lineal generalizado de Poisson (GLM-Poisson). Este modelo ampl√≠a el modelo de regresi√≥n de Poisson b√°sico para acomodar diferentes tipos de variables predictoras.

  • Si alguna variable independiente es cuantitativa, el modelo sigue denomin√°ndose modelo de regresi√≥n de Poisson.
